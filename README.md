# usafmcgportfolioanalysis
Analysed a US Consumer Staples portfolio using Python, covering Procter &amp; Gamble, Coca-Cola, PepsiCo, Walmart, and Costco. Calculated daily returns, annual returns, volatility, Sharpe ratio, maximum drawdown, and VaR using Yahoo Finance data (2015-2026) to evaluate portfolio performance and risk management.
